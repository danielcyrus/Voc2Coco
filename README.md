<br>
1-Copy all xml files into a directory<br>
2-run windows commnad prompt<br>
3-Goto xml directory and run "dir > files.txt"<br>
4-run the python file:<br>
  python Voc2Coco.py files.txt /<xmldirectory> output.json
 <br>
 <br>
